# Currency Converter

**Автор:** Богдан Павленко


## Описание

Currency Converter — это графическое приложение на Python для конвертации валют с использованием актуальных курсов из внешнего API. Приложение сохраняет историю всех конвертаций и позволяет экспортировать данные.

### Основные возможности

- **Конвертация валют** между любыми поддерживаемыми валютами
- **Актуальные курсы** из ExchangeRate-API (обновляются в реальном времени)
- **Выбор валют** из выпадающего списка (более 160 валют)
- **Перестановка валют** местами одной кнопкой
- **Автоматическое сохранение истории** в JSON файл
- **Просмотр истории** всех конвертаций в таблице
- **Экспорт истории** в отдельный JSON файл
- **Кэширование списка валют** для экономии трафика

## Получение API ключа

1. Перейдите на сайт [ExchangeRate-API](https://www.exchangerate-api.com/)
2. Нажмите "Get Free Key" (бесплатный тариф позволяет 1500 запросов в месяц)
3. Зарегистрируйтесь, указав email
4. Подтвердите email (потребуется подтверждение)
5. Скопируйте полученный API ключ
6. В коде замените `YOUR_API_KEY_HERE` на ваш ключ

### Альтернативные API (если основной не работает):

- **FreeCurrencyAPI**: `https://freecurrencyapi.net/`
- **CurrencyAPI**: `https://currencyapi.com/`

## Требования

- Python 3.6 или выше
- Библиотека `requests` (установка: `pip install requests`)
- Стандартные библиотеки: `tkinter`, `json`, `os`, `datetime`
